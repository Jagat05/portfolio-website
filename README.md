## About

Personal portfolio website showcasing my skills, projects, and experience as a Full Stack Developer. Built with modern web technologies and optimized for performance and SEO.

**Live:** [jagatjoshi.com.np](https://jagatjoshi.com.np)

## Features

- ✅ Modern, responsive design (mobile, tablet, desktop)
- ✅ Dark/Light theme toggle
- ✅ Dynamic project showcase with filtering
- ✅ Contact form integration (Formspree)
- ✅ Smooth animations and transitions
- ✅ SEO optimized (meta tags, structured data, sitemap)
- ✅ PWA ready with manifest.json
- ✅ Fast loading with optimized assets

## Tech Stack

**Frontend:**
- HTML5, CSS3, JavaScript (ES6+)
- Google Fonts (Poppins, Fira Code)

**SEO & Performance:**
- JSON-LD Structured Data
- XML Sitemap & robots.txt
- Optimized images & caching
